### Initialization
**Make sure you already have SymPy installed.**
Open terminal, go into the root path, run python3
```
python3
```
then import `PyCAS.py`
```python
from PyCAS import * 
```
if you don't wanna import it globally, you can do this:
```python
import PyCAS
```
